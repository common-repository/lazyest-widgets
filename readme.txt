== Description ==

This plugin adds new widgets to [Lazyest Gallery](http://wordpress.org/extend/plugins/lazyest-gallery/).

= included since version 0.6 of this plugin =
* Recent Images Widget
* Recent Images Slideshow
* Really Random Image Widget
* Really Random Image Slideshow Widget
* Latest Folders Widget

== Installation ==

0. Install, Activate and Configure Lazyest Gallery
1. Install using the WordPress plugin installer
2. Activate
3. Add widgets in the Appearance -> Widgets menu
